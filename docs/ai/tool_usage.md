# Tool Usage

This document defines tool usage guidance for the agency.

It describes how agents should use external tools without turning tools into the source of truth.

How To Use:

Read this file when wiring agents to automation, search, or validation tools.

## Tool Principles

- Use tools to execute or verify.
- Use docs to decide.
- Use schemas to validate.
- Use templates to start safe output.

## Tool Rules

- Tools should not override the documented workflow.
- Tools should not introduce new artifact names.
- Tools should not bypass review or approval steps.
