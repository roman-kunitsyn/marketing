# Retrieval

This document defines retrieval guidance for the agency.

Retrieval helps agents find the right documentation, schema, template, or dictionary entry for the current task.

How To Use:

Read this file when implementing search, indexing, or retrieval logic for the agent layer.

## Retrieval Rules

- Retrieve the most specific relevant source.
- Prefer source-of-truth documents over summaries.
- Retrieve schema and template pairs together when producing artifacts.

## Retrieval Targets

- Business docs
- Agency docs
- Marketing docs
- Content docs
- Design docs
- Platform docs
- Workflow docs
- Schemas
- Templates
- Dictionaries
