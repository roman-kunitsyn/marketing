# Facebook

This document defines the Facebook adaptation rules used by the agency.

Facebook is a current platform in the main docs.

How To Use:

Read this file when adapting content for Facebook-specific formats and publishing constraints.

## Supported Content

- Hero images
- Carousels
- Stories
- Testimonials
- Case studies
- Educational posts
- Offers

## Platform Rules

- Prioritize clarity and usefulness.
- Keep the message readable in feed and share contexts.
- Adapt captions for a slightly broader audience when needed.
- Keep platform formatting consistent with the content package.
