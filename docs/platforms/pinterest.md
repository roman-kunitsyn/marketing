# Pinterest

This document defines the Pinterest adaptation rules used by the agency.

Pinterest is a planned platform in the main docs.

How To Use:

Read this file when preparing visual search-oriented content for Pinterest.

## Supported Content

- Hero images
- Carousels
- Educational posts
- Offers

## Platform Rules

- Prioritize visual clarity and evergreen usefulness.
- Keep the composition searchable and useful as a reference.
- Adapt assets for discovery and saving behavior.
