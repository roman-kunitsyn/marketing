# LinkedIn

This document defines the LinkedIn adaptation rules used by the agency.

LinkedIn is a planned platform in the main docs.

How To Use:

Read this file when preparing content for LinkedIn-specific publishing.

## Supported Content

- Educational posts
- Case studies
- Testimonials
- Offers
- Hero images

## Platform Rules

- Prioritize professional tone and credibility.
- Keep message density moderate and value-driven.
- Adapt content to business and authority contexts.
