# TikTok

This document defines the TikTok adaptation rules used by the agency.

TikTok is a planned platform in the main docs.

How To Use:

Read this file when preparing short-form video content for TikTok.

## Supported Content

- Reels or short-form video equivalents
- Stories-style updates
- Educational snippets
- Offers

## Platform Rules

- Prioritize speed, clarity, and hook strength.
- Keep the message direct and easy to consume.
- Adapt visuals for motion-first delivery.
