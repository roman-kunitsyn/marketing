# Platforms

This folder defines platform-specific adaptation rules.

It explains how content should be adjusted for each supported platform without depending on execution tools.
Use it to capture platform audience, content constraints, best practices, and publishing considerations.

Current platforms in the main docs:

- Instagram
- Facebook

Planned platforms in the main docs:

- LinkedIn
- TikTok
- YouTube
- Telegram
- WhatsApp
- Pinterest

Use this folder for:

- platform overviews
- supported content types
- image sizes
- posting guidance
- captions and hashtags
- common mistakes

How To Use:

Read [README](../README.md) first, then use this folder to adapt content for a specific platform.
