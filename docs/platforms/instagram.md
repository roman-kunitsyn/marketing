# Instagram

This document defines the Instagram adaptation rules used by the agency.

Instagram is a current platform in the main docs.

How To Use:

Read this file when adapting content for Instagram-specific formats and publishing constraints.

## Supported Content

- Hero images
- Carousels
- Stories
- Reels
- Testimonials
- Educational posts
- Offers

## Platform Rules

- Prioritize strong visual hooks.
- Keep captions concise and scannable.
- Use hashtags only when they support discovery.
- Keep the asset aligned with the content package and creative package.
