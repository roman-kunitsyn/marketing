# WhatsApp

This document defines the WhatsApp adaptation rules used by the agency.

WhatsApp is a planned platform in the main docs.

How To Use:

Read this file when preparing content for WhatsApp-specific sharing or delivery.

## Supported Content

- Updates
- Offers
- Announcements
- Short supporting visuals

## Platform Rules

- Prioritize clarity and directness.
- Keep messages short and useful.
- Avoid overly complex layouts.
