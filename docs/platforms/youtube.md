# YouTube

This document defines the YouTube adaptation rules used by the agency.

YouTube is a planned platform in the main docs.

How To Use:

Read this file when preparing content for YouTube-specific publishing.

## Supported Content

- Educational videos
- Case study videos
- Explainer videos
- Offer videos

## Platform Rules

- Prioritize explanation depth and watchability.
- Keep the visual concept aligned with the topic.
- Adapt the message for longer-form viewing.
