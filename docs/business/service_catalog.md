# Service Catalog

This document lists the agency services in business terms.

It should be used to define what is offered, what inputs are required, and which artifacts are delivered.

How To Use:

Read this file when packaging an offer or scoping work. Use it with [Pricing](pricing.md) and [Contracts](contracts.md) to keep client-facing promises consistent.

## Service Categories

### Strategy Services

- Discovery and qualification
- Client interview
- Marketing brief creation
- Campaign strategy
- Content strategy

### Production Services

- Content package creation
- Copy package creation
- Creative package creation
- Image prompt package creation
- Publishing plan creation

### Delivery Services

- Review and QA
- Revision handling
- Final delivery
- Handoff support

## Standard Offer Structure

- Strategy-only engagement
- Full campaign build
- Monthly production retainer
- Audit and refinement sprint

## Service Inputs

- Business goals
- Brand profile
- Target audience
- Platform scope
- Content volume
- Approval constraints

## Service Outputs

- Named artifact set
- Delivery timeline
- Revision policy
- Handoff checklist
