# Image Sizes

This document defines the image size conventions used by the agency.

It gives a stable reference for designing assets that need to fit multiple platform formats.

How To Use:

Read this file when planning visual assets for platform adaptation.

## Image Size Principles

- Size should match the asset type.
- Size should support the intended platform output.
- Size should preserve readability and composition.

## Image Size Rules

- Define size expectations before creative production.
- Use consistent naming for size targets.
- Keep the size guidance independent from generator-specific settings.
