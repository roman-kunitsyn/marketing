# Image Assets

This document defines the reusable image asset layer used by the agency.

It describes the visual building blocks that can be referenced in creative packages and prompt work.

How To Use:

Read this file when identifying or naming reusable visual elements.

## Asset Types

- Hero image
- Carousel slide
- Story card
- Reel cover
- Testimonial card

## Asset Rules

- Use canonical asset names.
- Keep asset names stable across docs and schemas.
- Distinguish the asset itself from the layout or prompt used to create it.
