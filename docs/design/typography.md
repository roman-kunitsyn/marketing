# Typography

This document defines the typography guidance used by the agency.

It describes how text should be treated in campaign assets and supporting visuals.

How To Use:

Read this file when defining text hierarchy for creative work.

## Typography Principles

- Typography should support readability first.
- Typography should fit the brand tone.
- Typography should be usable across content formats.

## Typography Rules

- Define hierarchy clearly.
- Keep font choices stable within a campaign.
- Avoid mixing too many text styles in one asset.
