# Design

This folder defines the visual knowledge layer.

It covers composition, hierarchy, balance, contrast, typography, color, layout, and brand presentation.
The contents here should be independent of any specific image generator.

Use this folder for:

- design system rules
- composition guidance
- layout guidance
- typography
- color guidelines
- branding
- image sizes
- safe areas
- image assets

Design knowledge informs creative decisions and prompt generation, but it is not a prompt library.
