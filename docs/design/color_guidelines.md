# Color Guidelines

This document defines the canonical color guidance used by the agency.

It gives a reusable color direction for campaign assets and brand presentation.

How To Use:

Read this file when choosing or reviewing campaign colors.

## Color Principles

- Colors should support the brand and campaign mood.
- Colors should remain legible across formats.
- Colors should be usable in both content and creative packages.

## Color Rules

- Use a limited palette unless the campaign explicitly requires otherwise.
- Keep contrast readable.
- Keep named colors consistent with the brand profile and dictionaries.
