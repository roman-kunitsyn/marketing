# Layouts

This document defines the layout guidance used by the agency.

Layouts describe how content and visual elements are arranged in a creative asset.

How To Use:

Read this file when planning creative packages or image references.

## Layout Principles

- Layout should guide attention.
- Layout should support the message hierarchy.
- Layout should be reusable across asset types when possible.

## Layout Rules

- Keep layout descriptions simple and repeatable.
- Match layout choices to the content type.
- Avoid layout patterns that conflict with platform constraints.
