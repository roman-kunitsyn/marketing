# Composition

This document defines the composition guidance used by the agency.

Composition covers balance, hierarchy, contrast, spacing, and visual flow.

How To Use:

Read this file when reviewing visual direction or preparing creative instructions.

## Composition Principles

- Composition should make the message easy to scan.
- Composition should support the intended focal point.
- Composition should work with the selected layout and image size.

## Composition Rules

- Keep hierarchy explicit.
- Leave breathing room where the asset needs emphasis.
- Use composition to support the campaign objective.
