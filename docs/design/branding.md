# Branding

This document defines the branding layer used by the agency.

It describes how brand profile, voice, visual style, and colors come together in visual work.

How To Use:

Read this file when translating a brand profile into creative direction.

## Branding Inputs

- Brand profile
- Brand voice
- Visual style
- Color guidance
- Audience context

## Branding Rules

- Keep branding consistent across content and creative packages.
- Use the canonical brand terms from the core docs.
- Avoid overriding brand identity without explicit strategy input.
