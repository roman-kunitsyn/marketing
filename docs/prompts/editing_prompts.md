# Editing Prompts

This document defines editing prompt guidance for the agency.

Editing prompts are used to revise, refine, or normalize existing artifacts.

How To Use:

Read this file when creating prompts for QA-driven edits or revision support.

## Editing Prompt Inputs

- Existing artifact
- Review notes
- Quality criteria
- Target schema or format

## Editing Prompt Rules

- State the required change directly.
- Preserve valid structure unless structure changes are requested.
- Avoid changing the intent while editing the form.
