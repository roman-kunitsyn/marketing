# Image Prompts

This document defines image prompt guidance for the agency.

It covers prompts that produce or guide visual assets referenced by the creative layer.

How To Use:

Read this file when creating `image_prompt_package.json` content or image-generation instructions.

## Image Prompt Inputs

- Creative package
- Visual style
- Brand profile
- Layout guidance
- Image size
- Platform context

## Image Prompt Rules

- Make positive prompts specific and visual.
- Make negative prompts useful, not verbose.
- Keep style notes aligned with the design system.
- Keep prompts traceable to the creative package.
