# Prompt Patterns

This document defines reusable prompt patterns used by the agency.

Patterns help the team generate consistent prompts for repeated tasks.

How To Use:

Read this file when building new prompts or standardizing prompt libraries.

## Pattern Types

- Instruction plus context plus output format
- Role plus task plus constraints
- Review plus revision instructions
- Schema-driven generation

## Pattern Rules

- Keep patterns simple and reusable.
- Use canonical artifact names inside patterns.
- Prefer stable structure over clever wording.
