# Prompt Examples

This document defines example prompt structures for the agency.

It gives the team concrete reference points for writing prompt artifacts consistently.

How To Use:

Read this file when creating a new prompt or checking whether a prompt structure is consistent.

## Example Uses

- Generate a content package
- Generate a creative package
- Generate image prompts from a creative package
- Revise an artifact after review

## Example Rules

- Examples should reflect canonical naming.
- Examples should remain generic enough to reuse.
- Examples should not replace the main knowledge docs.
