# Prompts

This folder defines prompt engineering and compiled prompt artifacts.

Prompts are derived from business knowledge, content knowledge, and design knowledge.
They are not the source of truth for the business itself.

Use this folder for:

- prompt guidelines
- prompt structure
- prompt patterns
- image prompts
- copywriting prompts
- editing prompts
- prompt validation
- model profiles

Prompts are implementation artifacts for generation models, not business documentation.

How To Use:

Read [README](../README.md) first, then use this folder when generating prompts from the underlying knowledge layers.
