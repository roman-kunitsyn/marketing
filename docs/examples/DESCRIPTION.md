# Examples

This folder contains completed reference projects.

Examples are curated, approved, and reusable. They are used as high-quality references for future campaigns and as few-shot material for AI agents.

Use this folder for:

- completed project examples
- approved campaign artifacts
- anonymized reference work
- reusable patterns by industry

Each example should mirror the project structure closely enough that it can be compared against live work in `projects/`.
