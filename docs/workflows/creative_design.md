# Creative Design Workflow

This workflow defines how creative direction is produced.

It turns the content package into visual concepts, layouts, and image references.

How To Use:

Read this file when creating the visual side of a campaign package.

## Inputs

- `content_package.json`
- Design system
- Brand profile
- Visual style
- Image sizes

## Steps

1. Review the content package.
2. Select layouts and visual concepts.
3. Collect image references if needed.
4. Produce `creative_package.json`.
5. Validate that the visuals fit the campaign and platform context.

## Output

- `creative_package.json`

## Rules

- Creative design should stay independent from prompt syntax.
- Visual direction must remain reusable and platform-aware.
- The creative package should be clear enough for prompt generation.
