# Analytics Workflow

This workflow defines how published campaign results are tracked and reviewed.

It closes the loop between delivery and future campaign decisions.

How To Use:

Read this file when implementing reporting or performance analysis behavior.

## Inputs

- Published campaign data
- Platform metrics
- Campaign objective
- Success metrics

## Steps

1. Collect platform performance data.
2. Compare results to the campaign objective.
3. Record insights and issues.
4. Feed findings back into future planning.

## Outputs

- Performance summary
- Optimization notes
- Future planning inputs

## Rules

- Analytics should map to the approved campaign objective.
- Insights should be reusable for future strategy work.
- Analytics should not rewrite the original campaign record.
