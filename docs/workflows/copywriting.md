# Copywriting Workflow

This workflow defines how copy assets are produced.

It turns the content package and approved messaging into structured copy output.

How To Use:

Read this file when generating headlines, captions, CTAs, and related copy assets.

## Inputs

- `content_package.json`
- Brand voice
- CTA guidance
- Campaign messaging

## Steps

1. Review the content package.
2. Draft copy for each asset.
3. Align tone and CTA language.
4. Produce `copy_package.json`.
5. Validate copy quality and consistency.

## Output

- `copy_package.json`

## Rules

- Copy must match the campaign objective and audience.
- Keep the canonical naming consistent with the core docs.
- Do not mix creative layout instructions into copy output.
