# Agency

This folder defines the people-and-handoffs layer of the AI-First Marketing Agency.

It describes who owns each artifact, how artifacts move between roles, and how review happens.
It does not define business strategy or execution details.

Canonical roles in this project:

- Marketing Strategist
- Content Strategist
- Copywriter
- Creative Designer
- Prompt Builder
- Social Media Manager
- QA Reviewer

Use this folder for:

- role definitions
- artifact ownership
- communication rules
- handoff contracts
- review and quality guidelines

How To Use:

Read [README](../README.md) first, then use this folder to understand who owns each artifact and how handoffs work.
