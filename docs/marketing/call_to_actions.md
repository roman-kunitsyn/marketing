# Call To Actions

This document defines the CTA vocabulary used by the agency.

It helps the team keep conversion language consistent across campaigns and content formats.

How To Use:

Read this file when writing copy, reviewing conversions, or shaping funnel messaging.

## CTA Categories

- Learn more
- Contact us
- Book a call
- Download
- Sign up
- Request a quote
- Send a message

## CTA Rules

- CTAs should match the campaign goal and funnel stage.
- Use one primary CTA per artifact unless the format requires otherwise.
- Keep CTA language clear and action-oriented.
