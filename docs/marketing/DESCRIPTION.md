# Marketing

This folder defines the marketing domain model used by the agency.

It covers timeless marketing knowledge that informs strategy, content planning, and campaign design.
It should not mention execution tools or AI providers.

Use this folder for:

- marketing strategy
- campaign types
- campaign templates
- content strategy
- content pillars
- target audiences
- customer personas
- sales funnel
- customer journey
- brand voice
- visual style
- CTA guidance
- metrics
- best practices

Marketing knowledge is what the agency uses to make decisions before any content is produced.

How To Use:

Read [README](../README.md) first, then use this folder to make marketing decisions before creating content.
