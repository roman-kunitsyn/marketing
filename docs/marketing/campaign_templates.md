# Campaign Templates

This document defines the common campaign patterns used by the agency.

Templates turn campaign types into repeatable structures that can be reused across clients and platforms.

How To Use:

Read this file when turning a campaign type into a working structure for strategy or production.

## Template Components

- Campaign objective
- Audience
- Core message
- Content pillars
- Deliverables
- Publishing cadence
- Review checkpoints

## Template Rules

- Templates should be reusable across clients.
- Templates should not contain client-specific details.
- Templates should support the canonical artifact flow without hard-coding execution steps.
