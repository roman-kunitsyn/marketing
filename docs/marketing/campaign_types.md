# Campaign Types

This document defines the reusable campaign categories used in the agency.

Campaign types help the team choose strategy, content structure, and success metrics.

How To Use:

Read this file when classifying a client request or selecting a campaign format.

## Campaign Type Criteria

- Business objective
- Audience maturity
- Offer type
- Content volume
- Platform mix

## Examples

- Brand awareness
- Lead generation
- Product launch
- Promotion
- Evergreen nurture
- Event or announcement
- Authority building

## Rules

- Use campaign types consistently across briefs, strategies, and templates.
- Avoid inventing one-off labels when an existing campaign type fits.
- Keep the taxonomy stable so schemas and dictionaries stay reusable.
