# Target Audiences

This document defines the audience categories used by the agency.

It helps the team choose message framing, proof points, and content priorities.

How To Use:

Read this file when building marketing briefs, campaign strategy, or content strategy artifacts.

## Audience Criteria

- Role or segment
- Industry or context
- Need or pain point
- Buying intent
- Platform behavior

## Audience Rules

- Audience definitions should be specific enough to guide messaging.
- Do not collapse different intent levels into one audience label.
- Audience terminology should remain stable across documents.
