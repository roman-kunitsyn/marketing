# Visual Style

This document defines the visual style layer used by the agency.

It captures the visual direction that informs creative work before design execution begins.

How To Use:

Read this file when defining the look and feel of a campaign or preparing creative guidance.

## Style Dimensions

- Color direction
- Composition
- Texture
- Mood
- Contrast
- Image treatment

## Style Rules

- Visual style should be consistent with the brand and campaign objective.
- Visual style should remain usable across platforms and content types.
- Visual guidance should be descriptive enough for creative execution.
