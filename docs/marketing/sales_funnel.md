# Sales Funnel

This document defines the canonical funnel stages used by the agency.

It connects marketing goals to audience readiness and campaign messaging.

How To Use:

Read this file when mapping campaign content to the client buying journey.

## Funnel Stages

- Awareness
- Interest
- Consideration
- Decision
- Retention

## Funnel Rules

- Content should support the audience stage it targets.
- A campaign may serve multiple funnel stages, but the primary stage should be explicit.
- Funnel language should match the rest of the core docs.
