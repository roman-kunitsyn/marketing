# Brand Voice

This document defines the brand voice layer used by the agency.

It describes the tone and language rules that should remain stable across campaign assets.

How To Use:

Read this file when defining messaging, reviewing copy, or creating a marketing brief.

## Voice Dimensions

- Tone
- Formality
- Confidence
- Clarity
- Personality
- Vocabulary

## Voice Rules

- Voice should be consistent across all artifacts for a campaign.
- Voice should support the target audience and business goals.
- Voice guidance should be short, usable, and repeatable.
