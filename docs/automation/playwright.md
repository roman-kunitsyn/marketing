# Playwright

This document defines the Playwright automation layer for the agency.

Playwright is used for browser automation where publishing or QA tasks require UI interaction.

How To Use:

Read this file when implementing browser-based publishing or verification tasks.

## Responsibilities

- Browser interaction
- UI-driven publishing support
- Page verification
- Workflow validation support

## Rules

- Use Playwright only for execution concerns.
- Do not encode business rules in browser scripts.
- Keep browser steps aligned with the publishing workflow.
