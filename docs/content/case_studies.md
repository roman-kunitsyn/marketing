# Case Studies

This document defines the case study content type.

Case studies explain outcomes, process, and proof in a structured format.

How To Use:

Read this file when building credibility content or long-form proof assets.

## Case Study Purpose

- Demonstrate results
- Explain process
- Support authority and conversion

## Case Study Rules

- Case studies should include context, action, and outcome.
- Keep the structure reusable across clients.
- The content package should define the angle before writing begins.
