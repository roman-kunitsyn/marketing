# FAQ Posts

This document defines the FAQ post content type.

FAQ posts answer common questions in a concise, reusable format.

How To Use:

Read this file when handling objections or explaining core service details in content.

## FAQ Post Purpose

- Reduce friction
- Answer recurring questions
- Support clarity and trust

## FAQ Post Rules

- Use a single question per asset when possible.
- Keep answers direct and usable.
- Align the response with approved business and marketing language.
