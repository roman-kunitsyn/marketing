# Hashtags

This document defines the hashtag content layer used by the agency.

Hashtags are reusable discovery and organization tags attached to content assets where appropriate.

How To Use:

Read this file when preparing content for platform adaptation.

## Hashtag Rules

- Use only relevant hashtags.
- Keep hashtag sets consistent with the campaign topic and platform behavior.
- Do not treat hashtags as the core message.

## Hashtag Guidance

- Brand hashtags
- Campaign hashtags
- Topic hashtags
- Platform-specific hashtags
