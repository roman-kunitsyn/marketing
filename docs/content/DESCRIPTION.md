# Content

This folder defines the content production knowledge base.

It describes the content assets and content package building blocks used by the agency, independent of any platform.
A hero image, carousel, story, or reel exists here before it is adapted for Instagram, Facebook, or any other channel.

Use this folder for:

- content types
- content packages
- hero images
- single images
- carousels
- stories
- reels
- testimonials
- case studies
- educational posts
- FAQ posts
- offers
- captions
- hashtags
- calls to action

Content knowledge is about what is being created, not where it is published.

How To Use:

Read [README](../README.md) first, then use this folder to define content assets and packages before platform adaptation.
