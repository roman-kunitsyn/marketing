# Reels

This document defines the reel content type.

Reels are short-form motion assets built for attention, discovery, and platform-native engagement.

How To Use:

Read this file when planning or reviewing reel assets.

## Reel Purpose

- Drive attention
- Communicate a message quickly
- Support reach and engagement

## Reel Rules

- Reels should have a clear hook.
- Reels should be easy to adapt for a target platform.
- The content package should define the message and purpose before production.
