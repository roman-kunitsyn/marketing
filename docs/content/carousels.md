# Carousels

This document defines the carousel content type.

Carousels are multi-slide content assets used to present structured information in sequence.

How To Use:

Read this file when creating or reviewing carousel assets.

## Carousel Purpose

- Explain a topic step by step.
- Support educational or authority-building content.
- Hold attention through structured progression.

## Carousel Rules

- Each carousel should have a clear narrative structure.
- Slides should flow logically.
- The content package should define the core message before design work begins.
