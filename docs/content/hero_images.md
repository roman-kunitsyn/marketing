# Hero Images

This document defines the hero image content type.

Hero images are standalone visual assets used to anchor campaigns, posts, and landing-style content.

How To Use:

Read this file when planning or reviewing hero image assets.

## Hero Image Purpose

- Establish the visual hook.
- Support campaign messaging.
- Provide a reusable visual anchor for other assets.

## Hero Image Rules

- The image should be aligned with the brand and campaign intent.
- The image should be understandable without platform-specific context.
- Hero images should be described clearly in the content package.
