# Content Types

This document defines the canonical content types used by the agency.

It separates the content format from the platform it will eventually be published on.

How To Use:

Read this file when choosing which content asset to create. Use it with [Marketing](../marketing/DESCRIPTION.md) and [Platforms](../platforms/DESCRIPTION.md) to keep content planning consistent.

## Content Type Principles

- Content type describes what the asset is.
- Platform describes where the asset is used.
- A content type should be reusable across campaigns.

## Common Content Types

- Hero image
- Single image
- Carousel
- Story
- Reel
- Testimonial
- Case study
- Educational post
- FAQ post
- Offer post

## Rules

- Use the canonical term consistently across schemas, templates, and docs.
- Avoid mixing content type with publishing instructions.
- Keep content types independent from platform names.
