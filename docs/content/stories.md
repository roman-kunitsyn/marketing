# Stories

This document defines the story content type.

Stories are short-form sequential assets designed for fast consumption and platform-native behavior.

How To Use:

Read this file when planning story assets for a content package.

## Story Purpose

- Deliver quick updates
- Reinforce a message
- Support engagement and continuity

## Story Rules

- Story content should be concise.
- Story sequence should be easy to follow.
- The message should fit the campaign and audience context.
