# Educational Posts

This document defines the educational post content type.

Educational posts teach the audience something useful while supporting a campaign objective.

How To Use:

Read this file when planning teaching-oriented content.

## Educational Post Purpose

- Build trust
- Increase awareness
- Support expertise positioning

## Educational Post Rules

- Keep one primary lesson per post.
- Make the takeaway explicit.
- Match the post to the audience and funnel stage.
