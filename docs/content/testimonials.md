# Testimonials

This document defines the testimonial content type.

Testimonials are proof assets that support credibility and trust.

How To Use:

Read this file when using social proof in a content package.

## Testimonial Purpose

- Show proof
- Reduce buyer hesitation
- Support conversion-oriented campaigns

## Testimonial Rules

- Testimonials should be accurate and traceable.
- Use quotes or summaries consistently.
- Keep attribution and context clear.
